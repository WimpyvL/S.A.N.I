# `sani setup`

Initialize `~/.sani/sani.json` and the agent workspace.

Related:

- Getting started: [Getting started](/start/getting-started)
- Wizard: [Onboarding](/start/onboarding)

## Examples

```bash
sani setup
sani setup --workspace ~/.sani/workspace
```

To run the wizard via setup:

```bash
sani setup --wizard
```
